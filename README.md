# 🦜 YouTube & Website Summarizer using LangChain + Groq

A **Streamlit-based AI application** that summarizes content from **YouTube videos or websites** using **LangChain** and **Groq LLMs**.  
The app extracts text reliably (without `pytube`), processes it using modern LangChain APIs, and generates concise summaries in seconds.

---

## 🚀 Features

- ✅ Summarize **YouTube videos** using stable transcript extraction
- ✅ Summarize **web articles / blogs**
- ✅ Powered by **Groq LLMs** (fast & cost-efficient)
- ✅ Uses **modern LangChain (`invoke`) APIs**
- ✅ Clean Streamlit UI
- ✅ Robust error handling
- ❌ No `pytube` (avoids frequent YouTube API breakages)

---

## 🧠 Tech Stack

- **Python**
- **Streamlit** – UI
- **LangChain** – LLM orchestration
- **Groq** – LLM inference
- **youtube-transcript-api** – YouTube transcript extraction
- **Unstructured** – Website content parsing
